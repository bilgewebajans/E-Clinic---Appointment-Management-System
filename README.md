<h2 id="about">About</h2>

E-Klinik is a professional appointment management system developed for healthcare facilities and clinics. The system provides a smooth experience for both patients and management with features such as appointment creation and management, automatic reminders, and detailed reporting.

<h2 id="who-is-it-for">🎯 Who Is It For?</h2>

<h2 id="features">Features</h2>

### Core Features

- **Appointment Management**
  - Create and manage appointments online
  - Doctor-based availability check
  - Appointment status tracking (Pending, Confirmed, Completed, Cancelled)
  - Appointment history and reporting
  - Customizable appointment periods
  - Service management
  - 3 Different Appointment Calendar views (Weekly Schedule - List View - Appointment Calendar)
  - 2 Different Dashboard options (Service / Statistics)
  - One-Click Bulk SMS (Patients/Doctors)
  - Multi-Language (English / Turkish)
  - Automatic conflict checking

- **Doctor Management**
  - Doctor profile management
  - Working hours and availability calendar
  - Specialty areas and service definition
  - Doctor-based appointment view

- **Patient Management**
  - Patient registration and profile management
  - Patient history and notes
  - Patient-based appointment view
  - Detailed patient information

- **Automatic Notification System**
  - System notifications
  - Email notifications (SMTP support)
  - SMS notifications (API integration)
  - Appointment reminders (automatically scheduled reminders)
  - Customizable email/SMS templates
  - Notification history and log tracking

- **System Management**
  - Multi-role system (Admin, Doctor, Patient)
  - User authorization and permission management
  - System settings and configuration
  - Cache management
  - System monitoring and logs

- **Location Management**
  - City and district management
  - Patient and doctor location information

- **Reporting and Analytics**
  - Appointment statistics
  - Doctor performance reports
  - Patient statistics
  - Dashboard views

### Newly Added Features

- **2 Different Dashboard Screens**
  - **Service View Dashboard**: Service-based appointment tracking, daily schedule view and service performance analysis
  - **Statistics View Dashboard**: General statistic cards, appointment summaries and chart-based reporting
  - Default dashboard can be selected from organization settings

- **SMS - Email Log Records**
  - Centralized logging of sent SMS and email notifications
  - Tracking of delivery status, recipient and date information
  - Access via Monitoring Logs menu

- **Patient-Specific Notes**
  - Add, edit and delete notes on patient detail page
  - Attach service tags to notes (multiple selection)
  - Role-based authorization: Note creation (Admin, Doctor), viewing
  - Doctors can edit/delete their own notes

### User Interface

- Modern and responsive design
- Mobile-friendly interface
- Dark mode support
- User-friendly navigation
- Quick search and filtering

### Technical Features

- Asynchronous job queue (Redis technology)
- Scheduled reminder tasks (Laravel Scheduler)
- Email and SMS template system
- Cache optimization
- Secure authentication

<h2 id="usage">Usage</h2>

### Admin Panel

1. Log in with your admin account
2. Review the system overview from the Dashboard
3. View and manage your appointments
4. Set your doctors' working hours
5. Manage from Doctors, Patients and Appointments menus
6. Configure the system in the Settings section
7. User Guide
8. System Notifications (New Registration % New Appointment)

### Patient Panel

1. Log in or register with a patient account
2. Create an appointment from the Book Appointment page
3. View your appointments in the My Appointments section
4. Review appointment details
5. System Notifications (Appointment Confirmation)

<h2 id="technologies">Technologies</h2>

- **Backend**: Laravel 12.x
- **Frontend**: Bootstrap 5, JavaScript (Vanilla)
- **Database**: MySQL/MariaDB
- **Queue**: Redis
- **Cache**: Redis / File
- **Email**: SMTP
- **SMS**: API Integration
